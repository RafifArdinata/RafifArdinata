## 📦 Instalasi

Clone repository:

```bash
git clone https://github.com/username/nama-project.git
```

Masuk ke folder project:

```bash
cd nama-project
```

Install dependencies:

```bash
npm install
```

Jalankan project:

```bash
npm run dev
```
